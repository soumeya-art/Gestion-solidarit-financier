
# Système de Gestion de Solidarité Financière

## Description
Ce projet est une application web de gestion de solidarité financière développée dans le cadre d’un projet universitaire à l’Université de Djibouti.
L’application permet de gérer une caisse de solidarité où les membres peuvent cotiser, faire des demandes d’aide financière, et suivre les transactions. Les administrateurs peuvent superviser les contributions, valider les demandes et gérer les membres via une interface dédiée.

## Objectifs
- Automatiser la gestion de la caisse de solidarité
- Assurer la transparence des transactions
- Centraliser les données (membres, cotisations, aides)
- Faciliter le suivi des contributions et des demandes
- Offrir une interface simple et intuitive
